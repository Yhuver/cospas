== Description ==
Select default settings for image link when you upload/insert images in post or pages. When you insert a image in a post/page, by default, always it appears linked to media file.
With this plugin you will be able to stablish the link type by default for images which are inserted in post/pages without remembering select this option for each occasion.

For SEO or other causes, some users prefer **images with 'None' link**. Also, others users prefer that **images link to Attachment page** or directly **link to self file** .jpg, .png, .gif, etc.

Default Image Link let you choose between this links types for images:

* None
* Media File
* Attachment Page
* Custom URL

For each image, you will be able to change again the link type and the links of images that were inserted in post/pages previously will never be modified.

Also, this plugin can be useful for webs where publish multiple users that sometimes forget to select the correct link for each image uploaded/inserted in post or pages.

= Descripción en Español =
Selecciona el enlace por defecto que tendrán tus imágenes cuándo son subidas/insertadas en un artículo o página. Cuando insertas una imagen en un artículo/página, por defecto, siempre aparece enlazada al propio archivo que has subido.
Con este plugin podrás establecer el tipo de enlace por defecto que quieres que tengan las imágenes cuando son insertadas en artículos/páginas sin tener que acordarte de cambiarlo en cada ocasión.

Por motivos de SEO o cualquier otra causa, muchos usuarios prefieren que las **imágenes siempre aparezcan sin enlace**. También hay quien prefiere que las **imágenes enlacen a la correspondiente página de adjuntos de WordPress** o directamente **enlazar al correspondiente archivo** .jpg, .png, .gif, etc. 

El plugin Defaul Image Link te permite seleccionar entre los siguientes tipos de enlace por defecto para las imágenes:

* Ninguno; Las imágenes aparecerán por defecto sin enlaces.
* Archivo Multimedia: Las imágenes aparecen por defecto enlazadas a su correspondiente archivo .jpg, .png, .gif, etc.
* Página de Adjuntos: Las imágenes aparecen por defecto enlazadas a su correspodiente página de Adjuntos de WordPress.
* URL Personalizada: Enlace personalizado para cada imagen.

Para cada imagen se podrá volver a cambiar posteriormente el tipo de enlace y en ningún caso se modificarán los enlaces de las imágenes que ya han sido insertadas en artículos o páginas anteriormente.

Este plugin también puede ser de gran utilidad en webs en donde publican varios usuarios que en múltiples ocasiones olvidan revisar el enlace asignado a cada imagen que es insertada/publicada en los artículos o páginas.

== Installation ==
1. Upload "default-image-link" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to Settings Page and select prefered option.

= Instalación en Español =
1. Subir la carpeta "default-image-link" al directorio "/wp-content/plugins/".
2. Activar el plugins desde el menú "Plugins" en WordPress.
3. Ir a la Página de Ajustes y seleccionar la opción preferida.


== Frequently Asked Questions ==
= Can I disable the plugin after change default link for images? =
Of course! If you wish, after change default link for images, you can disable the plugin and enable it when you'll need change default image link egain.

= ¿Puedo desactivar el plugin después de cambiar el enlace por defecto para las imágenes? =
¡Por supuesto! Si lo deseas, después de cambiar el enlace por defecto para las imágenes, puedes desactivar el plugin y volver a activarlo cuando necesesites cambiar nuevamente el enlace por defecto para las imágenes.
